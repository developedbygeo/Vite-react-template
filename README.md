# Vite React Toolkit Tailwind Template

This templates provides the following toolchain:

- Vite
- React
- React DOM
- React Router DOM
- Redux Toolkit
- TailwindCSS
- Vite Plugin SVGR
- Prop-types
- ESLint + Airbnb plugin
- Prettier

<br>

## Installation

To install:

```bash
npm install
```

To run the dev server:

```bash
npm run dev
```
